## How to publish on GitHub Pages

1. Go to [github.com](https://github.com) and create a free account if you don't have one.
2. Click **New repository**.
3. Name it exactly: `yourusername.github.io` (replace with your GitHub username).
4. Set it to **Public** and click **Create repository**.
5. Upload all these files — drag and drop them directly into GitHub, keeping the folder structure (`css/` and `js/` folders).
6. Go to **Settings → Pages** → Source: **Deploy from branch → main → / (root)** → Save.
7. Your site will be live at `https://yourusername.github.io` within 2–3 minutes.

## Custom domain (optional)
If you buy `costella.africa` or `getcostella.com`:
1. Add a file named `CNAME` to the root folder containing just your domain name, e.g. `costella.africa`
2. Point your domain's DNS to GitHub Pages (A records: 185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153)
3. Enable HTTPS in GitHub Pages settings.

## To update the site
Edit any HTML file and re-upload it to GitHub. Changes go live in under a minute.
